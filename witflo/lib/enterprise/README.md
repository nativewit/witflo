## Core Features (Always Free)

The core Witflo application in `lib/core/`, `lib/features/`, and `lib/ui/` remains open source under MPL-2.0. This includes:

- End-to-end encryption
- Notes, notebooks, and tags
- Local-first architecture
- File-based sync
- All privacy and security features

## Business Model

We follow an **open core** model:
- **Core**: Open source (MPL-2.0) - always free
- **Enterprise**: Proprietary add-ons for business customers

This model ensures:
- Individual users get full privacy features for free
- Businesses can purchase additional team management features
- Development is sustainable long-term
- The core remains transparent and auditable
